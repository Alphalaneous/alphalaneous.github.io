What is GDBoard?

GDBoard is an advanced Geometry Dash Level Request Dashboard for Twitch!

With it comes many features such as many commands, extensive level filters, some easter eggs, and level analysis so you don't get any pesky bad levels.

Questions and Answers

Why should I use this?

GDBoard, contains a lot more fresh and helpful features to both you and the viewers. It has features to prevent levels againts the Twitch TOS using level analysis, which scans a level to check if the level has the image hack or vulgar language.

How does one request?

There are many ways to request a level with GDBoard, including simply typing the levl ID in chat! To find out more, do "!help !request" in your chat!

Can I block and filter levels?

You sure can! Using either the block button in the actions window to block, and the request settings to filter out levels you may not want being requested.

Chat Bot

Allow viewers to interact and request levels with ease. No more pesky forms or reading it from chat! I included a ton of commands for everyone! For example let's say a viewer wants to view the song from the current level, all they gotta do is "!song".

Command Settings

GDBoard allows for you to make commands do what you want, including custom commands! You can disable a command, make it mod only, set a cooldown, and allow it to send in whispers at ease!
