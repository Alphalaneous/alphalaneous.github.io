What is GDBoard?

GDBoard is an advanced Geometry Dash Level Request Dashboard for Twitch!

With it comes a brand new interface, themed off of the Xbox Game Bar, more chat commands, extensive level analysis to detect levels containing the image hack, and more!

Questions and Answers

Why should I use this?

GDBoard, contain's a lot more fresh and helpful features to both you and the viewers. It has features to prevent against ToS levels such as level analysis, which scans a level to check if it has the image hack or vulgar language.

How does one request?

There are many ways to request a level with GDBoard, including simply typing the levl ID in chat! To find out more, do "!help request" in your chat!

Can I block levels?

You sure can! Using either the block button in the actions window, or using "!block" in chat.

Chat Bot

Allow viewers to interact and request levels with ease. No more pesky forms or reading it from chat! I included a ton of commands for everyone! For example let's say a viewer wants to view the song from the current level, all they gotta do is "!song".

Redesigned Interface

See everything you need with the press of a button! GDBoard overlays your screen with all the information you need. Don't want something showing, just close it out. Wan't something to show even after closing the overlay, pin it!

Command Settings

GDBoard allows for you to make commands do what you want, eventually including custom commands as well. You can disable a command, make it mod only, and allow it to send in whispers!
