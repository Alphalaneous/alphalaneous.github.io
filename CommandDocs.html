Adding, Editing, and Deleting commands from chat

Command parameters are an optional way to specify cooldown, userlevel and aliases for a command

Example:

Alphalaneous: !commands add !schedule -ul=moderator -cd=5 @$(user), Alphalaneous streams sometimes.

This command can only be run by moderators or above and has a 5 second cooldown

Parameters:

Parameter Action
-ul / -userlevel sets the userlevel the command can be ran at, levels are everyone, subscriber, twitch_vip, moderator, and owner
-cd / -cooldown sets the command cooldown
-a / -aliases sets aliases of the command, comma separated
